# Tutorials

Each tutorial demonstrates a different aspect of soundscape analysis with Soundscapy.

If you are new to soundscape analysis, start with [About Soundscape Analysis](../background.md) for an overview of the core concepts before diving into code.

## Where to start

| I want to... | Go to |
| --- | --- |
| Understand the concepts behind the analysis | [About Soundscape Analysis](../background.md) |
| Run something immediately | [Quick Start](rendered/QuickStart.md) |
| Learn step by step, new to Python | [Getting Started](rendered/0_QuickStart_for_Beginners.md) |
| See the analytical approach behind the library | [Analysing and Representing Soundscapes](rendered/HowToAnalyseAndRepresentSoundscapes.md) |
| Work through a full assessment step by step | Tutorials 1–6 below, in order |
| Analyse audio recordings | [Binaural Audio Analysis](rendered/BinauralAnalysis.md) — requires `soundscapy[audio]` |
| Use the Soundscape Perception Index or CircE | [Soundscape Perception Index](rendered/4_Understanding_Soundscape_Perception_Index.md) — requires `soundscapy[r]` |
